## 🐹 서비스 소개

#### 🏥 위치 기반 병원 대기 현황 및 원격 예약
사용자의 현재 위치를 기반으로 주변 병원을 조회하고, 실시간 대기 인원을 확인하여 대기열이 적은 병원으로 즉시 원격 예약을 도와주는 플랫폼입니다.

#### 💊 근처 영업중인 약국 및 정보 조회
진료 후 가장 가까운 약국이나 야간 영업 약국을 확인하여 효율적인 의료 서비스 이용이 가능합니다.

#### ✨ 주요 기능
- **📍 실시간 위치 서비스**: LBS 기술을 활용한 내 주변 의료시설 탐색
- **⏱ 실시간 대기 현황**: 진료과목별 대기 인원 및 예상 대기 시간 제공
- **📅 스마트 예약**: 문진표 작성 및 원격 대기 등록 시스템
- **📂 의료 기록 관리**: 진료 이력, 처방전, 관련 문서 저장 및 열람



## 🐰 프로젝트 기획 및 설계

- **🎯 기대 효과**: 병원 무한 대기 현상 감소 및 사용자 중심의 효율적인 의료 환경 구축
- **🛠 설계 목표**: 위치 기반 데이터(LBS)와 실시간 데이터 처리를 통한 프론트-백엔드 간 원활한 통신
